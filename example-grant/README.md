# Example Grant

## Overview

This is an example grant repository. A simple Go program that prints text to the
console. It is meant to demonstrate the development process outlined in the
[Grants Guide](../README.md).

### How to run

To run the program, install Go from its [official website](https://golang.org/),
then run the following command from within the `example-grant` directory:

```bash
go run ./
```
